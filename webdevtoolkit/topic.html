HTML Basics

Category: HTML

Difficulty: Beginner

HTML stands for HyperText Markup Language. It provides the structure of a webpage by organizing content with headings, paragraphs, links, images, lists, forms, navigation areas, main sections, and footers.

Meta Tags

Category: HTML

Difficulty: Beginner

Meta tags provide information about a webpage to browsers, search engines, and social media platforms. They can identify the page description, author, character encoding, and viewport settings.

Forms

Category: HTML

Difficulty: Intermediate

HTML forms collect information from users through controls such as text fields, email fields, radio buttons, checkboxes, select menus, and submit buttons. Labels help users understand each form control.

CSS Basics

Category: CSS

Difficulty: Beginner

CSS stands for Cascading Style Sheets. It controls the appearance of HTML content, including colors, fonts, spacing, borders, backgrounds, alignment, positioning, and page layouts.

CSS Selectors

Category: CSS

Difficulty: Beginner

CSS selectors identify which HTML elements should receive a set of styles. Selectors can target element names, classes, IDs, attributes, states, and relationships between elements.

Flexbox

Category: CSS

Difficulty: Intermediate

Flexbox is a CSS layout system used to arrange elements in a row or column. It helps control alignment, spacing, direction, wrapping, and the distribution of available space.

Reset and Normalize CSS

Category: CSS

Difficulty: Intermediate

Reset and Normalize CSS reduce differences between browser default styles. A reset removes many default styles, while Normalize CSS keeps useful defaults and makes browser rendering more consistent.

JavaScript Basics

Category: JavaScript

Difficulty: Beginner

JavaScript adds behavior and interactivity to webpages. It can respond to user actions, change content, perform calculations, validate forms, and control how page elements behave.

Functions

Category: JavaScript

Difficulty: Beginner

Functions are reusable blocks of code that perform specific tasks. They can receive values through parameters and return results, helping programs stay organized and reducing repeated code.

Conditionals

Category: JavaScript

Difficulty: Beginner

Conditionals allow a program to make decisions. Statements such as if, else if, and else run different code depending on whether a condition evaluates to true or false.

The DOM

Category: JavaScript

Difficulty: Intermediate

The Document Object Model represents an HTML page as a collection of objects. JavaScript can select those objects and change their text, attributes, styles, classes, or position on the page.

Events

Category: JavaScript

Difficulty: Intermediate

Events are actions that happen in a browser, such as clicking a button, typing in a field, submitting a form, or loading a page. Event listeners allow JavaScript to respond to those actions.

Arrays

Category: JavaScript

Difficulty: Intermediate

An array stores multiple values in one variable. Each value has an index, and JavaScript can add, remove, update, search, filter, sort, or loop through the values in the array.

Array Methods

Category: JavaScript

Difficulty: Advanced

Array methods perform operations on array values. Methods such as forEach, map, filter, find, and sort help developers process information without writing the same looping logic repeatedly.

Objects

Category: JavaScript

Difficulty: Intermediate

A JavaScript object groups related information using property and value pairs. Objects can represent real items such as a book, student, product, recipe, or learning topic.

Sorting and Filtering

Category: JavaScript

Difficulty: Advanced

Filtering creates a new array containing values that meet a condition. Sorting changes the order of values based on rules such as alphabetical order, rating, date, price, or difficulty.

Mobile First Design

Category: Responsive Design

Difficulty: Intermediate

Mobile-first design begins with styles for smaller screens and then adds enhancements for larger screens. This approach encourages simpler layouts, efficient content, and improved mobile performance.

Media Queries

Category: Responsive Design

Difficulty: Intermediate

Media queries apply CSS rules when specific conditions are met, such as a minimum screen width. They allow layouts, navigation, typography, and spacing to adapt to different devices.

Accessibility and Design

Category: Accessibility

Difficulty: Advanced

Web accessibility helps people with different abilities use websites successfully. Accessible design includes semantic HTML, keyboard support, readable contrast, labels, alternative text, visible focus indicators, and clear page structure.
